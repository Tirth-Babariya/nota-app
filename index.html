One-time Supabase Setup
1
Go to supabase.com → create a free project
2
Open SQL Editor and run:
create table notes (
  id text primary key,
  user_id uuid references auth.users not null,
  title text default '',
  content text default '',
  tag text default 'none',
  pinned boolean default false,
  created_at bigint,
  updated_at bigint
);

-- Row Level Security
alter table notes enable row level security;
create policy "Users own notes" on notes
  for all using (auth.uid() = user_id);
3
Go to Authentication → Providers, enable Google, add your OAuth credentials (or use Supabase's built-in Google OAuth)
4
In Authentication → URL Configuration, add this page's URL to Redirect URLs
5
Paste your Project URL and anon key from Settings → API above
