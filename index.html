Quick Setup (5 min)
1
Create free project at supabase.com
2
SQL Editor → run:
create table notes ( id text primary key, user_id uuid references auth.users not null, title text default '', content text default '', tag text default 'none', pinned boolean default false, created_at bigint, updated_at bigint ); alter table notes enable row level security; create policy "own" on notes for all using (auth.uid() = user_id);
3
Authentication → Providers → Email — enable Magic Link
4
Authentication → Providers → Google — enable, paste Google Client ID + Secret (or skip for magic link only)
5
Authentication → URL Config — set Site URL + Redirect URL to your GitHub Pages URL
6
Settings → API — copy Project URL + anon key above
